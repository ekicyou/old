■このアーカイブについて
・どっとさくら、と呼んでやってください。
・このファイルは「伺か」用のghostファイルです。


************************************************************************
■動作環境
・「伺か」及び、その他互換環境にてご利用ください。
・Windows2000以上のOSで、SSP/CROW を利用することを推奨します。
　Windows95系の動作は保証しません。使用APIの関係で恐らく動きません。
・Pentium(586)の実行コードを含みます。
　このコードが実行できない環境では動作しません。


************************************************************************
■略歴
・オリジナル「任意」を元に、トークをリバースエンジニアリングし作成した
　「補完さくら」がオリジナル。
・日蔭氏により「偽さくら」→「偽・補完さくら」が作成される。
・駅長が引き継ぎ、「．さくら」仮リリース     （Version 0.00）
・IfGhost[．さくら]、画像を偽補完ベースに    （Version 0.20）
・栞エンジンをPHP5ベースの物に差し替える     （Version 1.00）


************************************************************************
■著作権について
画像、トークデータの著作権については、「．さくら」配布サイトを
参照ください。

ＡＩエンジンについて、PHP5本体以外の部分については駅長が著作権を
有します。NYSLライセンス(http://www.kmonos.net/nysl/)が適用されます。
再利用、改良についてこちらから制約をかけることはありません。
